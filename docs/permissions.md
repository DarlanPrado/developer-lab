## Roles globais

### Admin

Controle total da plataforma.

### User

Pode acessar recursos autorizados.

## Roles por workspace

| Role        | Permissões                                              |
|-------------|---------------------------------------------------------|
| Owner       | Gerenciar workspace, alterar recursos, convidar usuários |
| Maintainer  | Gerenciar configuração, executar deploys                |
| Developer   | Abrir terminal, visualizar logs, reiniciar ambiente     |
| Viewer      | Apenas visualização                                     |
