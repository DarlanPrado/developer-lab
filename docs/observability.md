## Métricas iniciais

- CPU
- Memória
- Uptime

**Fonte:** Docker Stats API

## Logs

Interface de logs com filtro por nível (info/warn/error), busca por texto e scroll infinito, sem necessidade de abrir terminal.

## Dashboard de uso

Visão consolidada de CPU, memória e disco consumidos por todos os workspaces e shared resources, útil para o admin planejar capacidade.
